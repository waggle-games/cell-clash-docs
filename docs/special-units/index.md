## 능력치 보는 법

각 세포 페이지의 능력치 표는 1~5 별 스케일로 다른 세포와 상대적으로 비교한 값입니다.

- **공격력** — 한 번의 공격으로 입히는 피해량
- **체력** — 자체 HP.
- **이동속도** — 평상시 이동 속도
- **사정거리** — 공격이 닿는 거리 또는 효과 반경
- **공격속도** — 공격 쿨다운이 얼마나 빠른지

별이 많다고 무조건 좋고, 별이 적다고 약점인 건 아니에요. 별이 적은 능력치는 다른 강점을 위해 의도적으로 양보한 부분입니다. (ex. 포격 세포는 이동속도 ★를 내주고 사정거리 ★★★★★★를 확보했어요.)

## 역할 태그

- **공격형 (⚔️)** — 적을 직접 처치하는 데 기여하는 세포
- **방어형 (🛡️)** — 충격을 받아 내거나 진격을 방해하는 세포
- **지원형 (🌿)** — 군집을 강화하거나 자원 수집 · 적 통제로 흐름을 바꾸는 세포
